# MoviesNews
This repository is for my new app called MoviesNews which give the user updated news about movies including popular movies, upcoming movies, and top rated movies.

Kotlin
---
MVVM and Repository pattern
---
Coroutines
---
Room
---
LiveData
---

---
JetpackNavigation
---
Retrofit
---
Gson
---
Picasso
